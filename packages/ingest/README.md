# @lodestone/ingest

Repository parser, graph builder, embedder runtime, clusterer, skill emitter,
watcher, and SQLite writer for Lodestone.

Friend releases publish two profiled ingest tarballs:

- `lite` bundles the Snowflake 384d embedder.
- `full` bundles the Nomic 768d embedder.

Both profiles run locally and do not upload source code.
