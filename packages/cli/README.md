# @lodestone/cli

Command-line interface for Lodestone friend installs and local project
management.

The friend installer uses this package to run `lodestone init`, write the
project-local MCP configuration, and expose daily commands such as `status`,
`reindex`, `doctor`, and `uninstall`.

For end-user documentation, see the packaged `docs/` directory or
https://lodestone.cmndi.ai/docs/.
